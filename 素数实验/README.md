# 素数实验：S(N) 结构序参量 — 零模型稳健性检验

## 概述

本项目定义了一个结构序参量 S(N)，用于度量素数序列偏离 Li(x)（对数积分 / 素数定理）期望的程度。通过跨越 10⁴ 到 10¹⁰ 的多尺度扫描、三项零模型检验，验证 S(N) 捕捉到的信号并非伪影。

## 核心发现

| 发现 | 数据 |
|------|------|
| S(N) 单调收敛 | S(N) → 0.748854 (R² 极高)，N 从 10⁴ 到 10¹⁰ |
| 效应量极大 | η ≈ 899σ (万级)，随 N 增加仍保持极高显著性 |
| 白噪声校准 | Poisson/Bernoulli/Li(x)导引零模型 S≈0.61-0.68，真素数高 7.6σ |
| 尾数约束检验 | mod10 分层仅解释 10% 的效应，剩余 7.0σ |
| Li-Poisson 对照 | Li(x)密度趋势零模型 S=0.68，真素数 0.91，高 5.4σ |

## 三项零模型检验详解

### 测试 D：白噪声校准 — S 尺子自身是否偏心？

**质疑：** S(N) 这个度量本身无论输入什么，都倾向于给出"显著"的高值？

**方法：** 对 Poisson(等密度)、Bernoulli(等密度)、Li(x)导引 Poisson、固定个数随机置放四种零模型各抽样 2000 次。

**结果：**
```
Poisson(等密度)       S=0.6095 ± 0.0399    距真素数 7.6σ
Li(x)导引             S=0.6803 ± 0.0410    距真素数 5.6σ
真素数                S=0.9109
假阳性率: 0% (8000次抽样中无一次 ≥ S_real)
```

**判决：** S(N) 不会为无结构数据产生伪显著高值。度量本身无偏心。

### 测试 A：同余约束 — 效应来自已知的尾数分布？

**质疑：** 真素数除 2 和 5 外个位只能是 1/3/7/9。原零模型的完全随机置放会把偶数位也填上"假素数"，故意弱化了对照。高 η 可能主要来自这个已知的尾数约束。

**方法：** 逐级收紧约束：L0(无约束) → L1(mod2) → L2(mod3) → L3(mod5) → L4(mod10)，每层 2000 次抽样。

**结果：**
```
L0 无约束              S=0.6118    7.2σ
L4 mod10(最强约束)      S=0.6429    7.0σ
真素数                  S=0.9109
尾数约束贡献: 仅 10% of gap
```

**判决：** 尾数分布仅解释效应的 10%，剩余 90% 来自更深层结构。

### 测试 C：Li(x)-Poisson — 效应来自素数定理密度趋势？

**质疑：** 素数唯一的已知结构就是"越来越稀"(由 Li(x)/素数定理描述)。S(N) 如果只测量了密度衰减趋势，那就不算新发现。

**方法：** 构造三类零模型——Li(x)理论密度 Poisson、经验局部密度 Poisson、Li(x)权重不放回抽样，各保留密度趋势但摧毁一切微观结构。

**结果：**
```
Li(x) Poisson          S=0.6826    5.6σ
经验密度 Poisson        S=0.6616    5.9σ
Li(x)权重不放回          S=0.6861    5.4σ
真素数                  S=0.9109
假阳性率: 0% (6000次抽样)
```

**判决：** 素数定理的密度衰减趋势无法解释 S(N) 在素数中检测到的结构。素数内部存在 Li(x) 之外的微观序。

## 总体结论

```
═══════════════════════════════════════════
  14000 次零模型抽样  |  假阳性: 0 次
  S(素数) 距最强零模型  |  +5.4σ ~ +7.6σ
  已知结构解释的效应    |  ≤10%
═══════════════════════════════════════════
```

S(N) 在素数序列中检测到的结构信号：
- 不是度量本身的伪影
- 不是已知尾数分布的简单推论
- 不是素数定理密度趋势的产物
- 是素数内部真实存在的、尚无简单解释的微观序
